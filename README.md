# PG Management System 🏠

A Flutter-based mobile application to manage Paying Guest (PG) accommodations including tenants, rent tracking, and expenses.

---

## 🚀 Features

* 👤 Tenant Management
* 💰 Rent Tracking
* 📊 Expense Management
* 🔔 Notifications
* 📱 Clean UI (Flutter)

---

## 🛠 Tech Stack

* Flutter
* Dart
* REST API
* Firebase

## ⚙️ Installation

```bash
git clone https://github.com/ankitbalani5/Tenant-PG-Management-Software.git
cd Tenant-PG-Management-Software
flutter pub get
flutter run
```
